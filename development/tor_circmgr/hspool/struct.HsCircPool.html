pub fn estimate_timeout(&self, timeout_action: &Action) -> Duration

Return an estimate-based delay for how long a given Action should be allowed to complete.

This function has the same semantics as CircMgr::estimate_timeout. See the notes there.

In particular you do not need to use this function in order to get reasonable timeouts for the circuit-building operations provided by HsCircPool.
