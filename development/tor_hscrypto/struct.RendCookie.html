Struct tor_hscrypto::RendCookie

source · 
pub struct RendCookie(_);
Expand description

An opaque value RENDEZVOUS_COOKIE used at a rendezvous point to match clients and services.

See rend-spec-v3 s4.1.

The client includes this value to the rendezvous point in its ESTABLISH_RENDEZVOUS message; the service later provides the same value in its RENDEZVOUS1 message.
